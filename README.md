# dotfiles
My Dotfiles

## Dependencies

### Linux

update apt-get

```
$ sudo apt-get update
```
  
install git

```
$ sudo apt-get install git
```
  
install vim

```
$ sudo apt-get install vim
```
  
install zsh and set as the default shell

```
$ sudo apt-get install zsh
$ chsh -s $(which zsh)
```
